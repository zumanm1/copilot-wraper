# Plan: Fix 3 Webpages ŌĆö piplinetask, task-completed, alerts

## Root Cause Analysis

### Task 1: `/piplinetask` ŌĆö Missing dynamic refresh controls
- **Current state**: Has a refresh button and 5s/15s auto-refresh, but NO user-selectable refresh interval
- **Issue**: User wants configurable refresh: 1s, 15s, 30s, 1min, 2min (default), 15min, 30min
- **Backend**: `/api/task-pipelines` endpoint works correctly (line 7240-7400 in app.py)
- **DB**: Reads from `task_runs`, `task_definitions`, `task_alerts`, `task_step_results`, `task_feedback_events`, `session_manager_sessions`
- **Flow**: Browser ŌåÆ C9b FastAPI ŌåÆ SQLite ŌåÆ render pipeline items with step flow, timeline, trace, recovery

### Task 2: `/task-completed` ŌĆö Missing dynamic refresh controls
- **Current state**: Has refresh button and 15s auto-refresh, but NO user-selectable interval
- **Issue**: Same as piplinetask ŌĆö needs configurable refresh intervals
- **Bug**: Duplicate `var lastExportData = [];` (line 134-135), duplicate event listeners for export buttons (line 309-312), duplicate `renderItem` grid section (lines 244-260)
- **Backend**: `/api/task-completed` endpoint works (line 7128 in app.py)
- **DB**: Reads `task_runs` (terminal statuses), `task_definitions`, `task_alerts`, `task_feedback_events`

### Task 3: `/alerts` ŌĆö Data not loading
- **Current state**: Shows header, filters, stats bar (all zeros), "Loading runtimeŌĆ”" ŌĆö but no alert cards
- **Root cause**: The alerts page template was rewritten in the previous session but has a **JavaScript syntax error** on line 653:
  ```js
  '<td><input type="checkbox" class="alert-select-cb" data-id="' + esc(a.id) + '" ' + (isSelected ? 'checked' : ') + '></td>'
  ```
  Missing closing quote: `'checked' : ')` should be `'checked' : '')`
  This causes the entire IIFE to fail silently ŌĆö `loadAlerts()` never executes, no data loads.
- **Backend**: `/api/alerts` endpoint works correctly (line 7188-7223 in app.py)
- **DB**: Reads `task_alerts` JOIN `task_definitions` JOIN `task_runs`
- **Flow**: Browser ŌåÆ C9b FastAPI ŌåÆ SQLite JOIN ŌåÆ return alerts array
