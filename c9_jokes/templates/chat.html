Select an agent and send a prompt to begin.
How it works

Your prompt is sent to POST /api/chat on this C9 service, which proxies it to C1 ({{ c1_url }}/v1/chat/completions) with the selected agent's X-Agent-ID header. C1 routes the request through C3 (browser-auth cookies) to Microsoft Copilot. Each agent ID maintains a separate backend session. All exchanges are logged in the C9 SQLite DB (see Logs).
