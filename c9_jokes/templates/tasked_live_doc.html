📋 Tasked Live Documentation
Tasked Types
▶ output
LLM produces readable text or data. Goes to C1b (chat) or sandbox.
⚡ alert
Evaluates condition (JSON trigger). Fires alert when {"triggered":true}.
⚙ action
Runs shell commands on C12b sandbox. Returns stdout/exit_code.
🔗 hook
Triggers external systems / generates webhook payloads.
✦ combined
All types: output + alert + action in one run.
🖥 sandbox
mode=sandbox sends commands directly to POST C12b:8210/exec.
Workflow Steps (per task)
  • trigger — marks start, records timestamp
  • chat / sandbox — executes LLM or shell command
  • alert — fires alert record if trigger condition met
  • complete — closes run, sets terminal_reason
C12b Sandbox Flow (action mode)
C9b → POST :8210/exec
  {command: "..."}
← {stdout, stderr,
   exit_code, session_id}
