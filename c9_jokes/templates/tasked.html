Tasked Recurring or manual Chat + Agent + Multi-Agent + multi-Agento Alerts integration

Tasked Orchestrator

Tasked is a separate orchestration page in C9. It stores recurring or one-off tasks, keeps editable templates, decides which execution surface to use, and raises alerts when a trigger condition is met. Background recurring execution currently runs chat tasks directly. agent, multi-agent, and multi-Agento tasks are saved here and launched into their dedicated pages when you run them.
Task orchestrationThe doer: picks the execution path and holds the saved definition.
Task plannerThe order of work, cascading actions, and tab-to-tab context handoff.
Task timerThe recurrence interval, active state, and next-run scheduling window.
Task executorThe exact prompt or code-oriented action executed by chat, agent, or team mode.
Task alert generatorRaises an alert on the Alerts page when the trigger condition is satisfied.

Execution Roles

These roles are editable through the builder fields below. The templates prefill them for the example flows you gave.

Task Orchestrator

Selects the mode, keeps the saved task, and decides whether the task runs in background chat mode or launches a larger interactive workspace.

Task Planner

Defines the step order, including cascading checks like detect email, inspect attachment, then check SharePoint.

Task Timer

Controls recurring frequency, active state, and next-run timing. Use it for every-10-minutes flows and manual one-off launches.

Task Executor

Runs the actual prompt or code-oriented action. This is where the detailed task instruction lives.

Task Alert Generator

Builds the alert payload that becomes visible on the Alerts page with trigger, summary, and extracted details.

Editable Templates

These templates are editable. Click Load template to prefill the builder, then change the prompts, schedule, tabs, or trigger logic. The current templates cover the weather example, Gmail/Outlook sender watch, SharePoint file watch, Outlook alert watch, and the combined Outlook + SharePoint flow.
Templates are DB-backed. Built-in and user-created templates can both be loaded here; archiving removes them from the active picker.

Tasked Builder

Use this builder for recurring or non-recurring tasks that combine chat, agents, multi-agent, or multi-Agento. Keep Tabs required and Context handoff explicit when you want one tab to gather data and another tab to continue the flow.
Saved rows are editable. The saved task definition is the source of truth, not the original template.

Traceability

Select or save a Tasked row to inspect its unique trace ID, latest run, latest alert, and role-by-role visibility from orchestration through alerts.
